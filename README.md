# Fish Tank Poker

Fish Tank Poker は、海外カジノの $2/$5 キャッシュゲームで勝てる下地を作るためのポーカートレーニングアプリです。

単なるポーカーゲームやGTOクイズではなく、ライブキャッシュで長期的に勝つための判断力、レンジ感覚、フォールド力、バリュー最大化を鍛えることを目的にしています。

## 起動方法

`fish_tank_poker.html` をブラウザで開きます。

## 現在の主な機能

- リングゲーム練習
- フロップトレーニング
- トーナメント局面別トレーニング
- 自動評価と判断分析
- 壁打ち用プロンプト生成
- 回帰検査

## 開発メモ

旧ファイル名 `holdem_trainer.html` から、アプリ名に合わせて `fish_tank_poker.html` を正ファイルに変更しています。
